---

.. toctree::
  ../speakers/speaker_32

- Abstract:
Zeljko will tutor us on how he built Python 3.4.3 from source and changed the interpreter grammar to listen to a different language ( namely Serbian/Cyrilic ).

The presentation will display the power of the Python interpreter and its flexibility; its potential use is to teach young people the basics of programming in a modern object-oriented language in countries where English language is not as prevalent.
